Principais Formatações

Uma palavra pode ter forma e significado, e a semântica dá mais valor ao significado,
ou seja, no html5 as tags são mais significativas, quando se escreve um endereço
usando a tag ADDRESS é possível visualizar esse enderço no maps, diferente de usar
a antiga tag U do html4 que só tinha formato visual, não usual.

Negrito/Destaque

Nesta frase temos um termo em negrito usando a tag B (não semântica).

Nesta frase temos um termo em destaque usando a tag STRONG (semântica).

Itálico/Ênfase

Nestra frase temos um termo em itálico usando a tag I.

esta frase temos um termo em itálico usando a tag EM.

Texto Marcado

Nesta frase temos um texto marcado (grifado) usando a tag MARK.

Texto pequeno e grande

Podemos criar um texto com fonte pequena usando a tag SMALL, e com a
fonte grande usando a tag BIG (QUE JÁ ESTÁ ABSOLETA).

Texto deletado

Podemos criar um texto excluído usando a tag DEL (é usado para indicar
que o texto deve ser lido, mas não considerado).

Texto inserido/grifado

Podemos escrever um texto destacado usando a tag INS.

Também existe o sublinhado usando a tag U (Não semântica).

Texto sobrescrito e subscrito

Para inserir números elevados, usa-se a tag SUP = X20/7 + 3

E para inserir números não elevados rsrsrsrsrs usa-se a tag SUB = H2O.
