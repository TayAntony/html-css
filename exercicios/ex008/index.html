Outras formatações

Código fonte/Pré formatação

Nesta frase temos o uso do comando CODE, que deixa as letras 'monoespaçadas'.

Esse comando é usado para deixar a letra no formato original da programação, por exemplo : 

            
    num = int(input('Digite um número'))
    if num % 2 == 0:
        print(f'O número {num} é PAR')
    else:
        print(f'O número {num} é ÍMPAR')
    print('Fim do programa')

Citações Simples

Para criar uma citação simples (uma linha aprox.), usa-se o comando Q, por exemplo:

Como diria o pai de um amigo: o computador é um burro muito rápido

Citações Completas

Para fazer citações mais longas (em bloco), usa-se o comando BLOCKQUOTE (que também
dá um récuo na citação), junto com o comando CITE (opcional) seguido de =, por exemplo:

Segundo Jeff Noble, no seu livro HTML para Leigos:

A diferença entre elementos inline e um bloco de texto é importante. Os elementos HTML
neste capítulo descrevem os blocos de texto.

Abreviações

Na frase a seguir teremos um exemplo do uso do comando ABBR, que é utilizado (não
necessariamente com o JS) para mostrar o significado de palavras abreviadas:

Estou estudando HTML e CSS, e definitivamente ESTOU AMANDO!
