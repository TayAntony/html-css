Usando links externos

Os links permitem a ligação de vários pontos dentro da World Wide Web,
e por isso é importante saber usá-los, especialmente os hyperlinks que é a
interação usada para te levar em outra página. É possível criar um hyperlink
usando a tag A (âncora), e colocando o link escolhido dentro das aspas:

Não se esqueça de conferir os exercícios no GitHub

Porém, quando o usuário clicar ele será direcionado para o link novo,
sobrepondo/fechando a janela do seu site, para evitar isso se usa o comando
TARGET="_BLANK" (abrir uma nova janela em branco) e REL="EXTERNAL",
que significa abrir o link (externo) numa aba nova.

Outros tipos de links

Além de next, prev e external também existe o link author, que indica que é
um link para o site do autor do artigo atual.

Outros tipos de Target

Além do target blank (que abre uma nova aba), também existe o target self
(abre na própria página html), target top (é aberta no frame de maior tamanho,
chamado "principal", carregando a página por toda a janela do navegador) e o
target parent (que ainda nao tenho certeza, igual o top)
