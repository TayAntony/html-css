Imagem Dinâmica

Diferentes dispositivos têm diferentes tamanhos de tela, e consequentemente a imagem do seu site precisa ser adaptada para esses tipos de tela, nesse exercício vamos aprender a usar a tag PICTURE, que junto com a tag SOURCE MEDIA TYPE delimita os tamanhos de imagens a serem carregadas para que não exista barra de rolagem lateral.

De maneira geral, o MAX-WIDTH delimita o tamanho máximo quea página terá para aparecer a imagem, por exemplo: se o tamanho máximo é 750px a imagem que vai carregar é a média, mas a partir do momento que for menor que 750px será a imagem pequena
