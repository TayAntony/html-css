Inserindo vídeos hospedados localmente

Este vídeo está hospedado no meu próprio servidor, e para inseri-lo basta usar a tag VIDEO (tirando src e colocando SOURCE SRC no lugar), e reduzir o tamanho com o comando WIDTH (não reduz o tamanho do arquivo), seguido do comando CONTROLS (assim como com os áudios), tudo dentro da tag de abertura <VIDEO>.

Sinto muito! Seu navegador não tem compatibilidade com reprodução de vídeos.

Inserindo vídeos do YouTube

Inserir vídeos do youtube é uma maneira muito rápida e prática, já que limita o uso de dados no seu site (diminui os gastos), e faz com que, se o cliente também for dono do vídeo, receba os lucros do youtube. E para fazer isso, basta procurar o vídeo, clicar em compartilhar e escolher a opção INCORPORAR, que automaticamente cria a tag necessária para colocar o vídeo no site, em seguida basta copiar a tag no seu programa!

Inserindo vídeos do Vimeo

Dá para fazer o mesmo com o site Vimeo (porém ele pode travar caso a conexão internet não seja boa), e tem muito mais opções de alteração (cor, nem do autor, etc).
